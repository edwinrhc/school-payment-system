# Frontend – School Payment System

Frontend desarrollado con **Angular 19** siguiendo buenas prácticas modernas
(Standalone Components, CLI local por proyecto).

Este proyecto **NO requiere Angular CLI instalado globalmente**.

---

## Requisitos

- Node.js **18+ o 20+**
- npm **9+**
- Navegador moderno (Chrome recomendado)

Verificar versiones:
```bash
node -v
npm -v
```

## Instalación

Clonar el repositorio e instalar dependencias:

```bash
npm install
```
## Servidor de desarrollo
Forma recomendada (CLI local)

```bash
npm run start
```
## Nota importante sobre Angular CLI

Este proyecto utiliza Angular CLI local (por proyecto).

````aiignore
npx ng serve
````
o
````aiignore
npm run ng -- generate component components/example
````
